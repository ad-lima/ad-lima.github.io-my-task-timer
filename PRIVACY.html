🔒 Política de Privacidade - My Task Timer

Última atualização: Janeiro de 2025
Versão: 1.0

📋 Introdução

O My Task Timer é uma extensão do Chrome projetada com foco na privacidade do usuário. Esta política explica como coletamos, usamos e protegemos suas informações.

📊 Dados que Coletamos

A extensão armazena os seguintes dados localmente no seu dispositivo:

  • Projetos e Tarefas: Nomes, descrições e tempos registrados
  • Configurações de Hidratação: Intervalos e preferências de lembretes
  • Preferências do Usuário: Tema, configurações de interface
  • Dados de Onboarding: Status de conclusão do tutorial
  • Analytics Básicos: Contadores de uso para melhorar a experiência

🚫 Dados que NÃO Coletamos

  • Informações pessoais identificáveis
  • Dados de navegação ou histórico
  • Informações de localização
  • Dados de outros sites ou aplicações
  • Informações de contato ou comunicação

💾 Como Armazenamos os Dados

Todos os dados são armazenados exclusivamente no seu dispositivo usando:

  • Chrome Storage API: Para configurações e preferências
  • IndexedDB: Para projetos, tarefas e dados de tempo
  • Local Storage: Para preferências de tema
🔐 Importante: Nenhum dado é enviado para servidores externos ou compartilhado com terceiros.

🔄 Compartilhamento de Dados

NÃO compartilhamos seus dados com terceiros. Especificamente:

  • ❌ Não vendemos dados
  • ❌ Não usamos para publicidade
  • ❌ Não fazemos tracking entre sites
  • ❌ Não enviamos para analytics externos
  • ✅ Todos os dados permanecem no seu dispositivo

🔧 Permissões Explicadas

storage

Para salvar suas tarefas, projetos e configurações localmente.

alarms

Para criar lembretes de hidratação em intervalos regulares.

notifications

Para exibir notificações de lembretes de hidratação.

tabs

Para detectar quando você está ativo e pausar/retomar o cronômetro automaticamente.

📤 Exportação de Dados

Você pode exportar todos os seus dados a qualquer momento:

  • Clique no botão "Exportar" na extensão
  • Os dados são salvos como arquivo Excel (.xlsx) no seu computador
  • O processo é completamente local, sem envio para servidores

🗑️ Exclusão de Dados

Para remover todos os dados:

  1. Vá para chrome://extensions/
  2. Encontre "My Task Timer"
  3. Clique em "Remover"
  4. Ou use a função "Limpar Dados" na própria extensão

🔄 Atualizações da Extensão

Quando a extensão é atualizada:

  • Seus dados locais são preservados
  • Nenhuma informação é enviada durante o processo
  • A política de privacidade pode ser atualizada (notificaremos sobre mudanças significativas)

⚖️ Conformidade

Esta extensão está em conformidade com:

  • GDPR (Regulamento Geral de Proteção de Dados)
  • CCPA (California Consumer Privacy Act)
  • Políticas do Chrome Web Store
